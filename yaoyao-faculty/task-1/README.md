### 任务[描述](http://ife.baidu.com/course/detail/id/97)    
在页面中实现一个输入框与按钮，要求点击验证按钮后，对输入框中内容进行格式校验，并在其下方显示校验结果
![示例图片](http://7xrp04.com1.z0.glb.clouddn.com/task_2_29_1.jpg)   

#### 校验规则：
1. 字符数为4~16位  
2. 每个英文字母、数字、英文符号长度为1
3. 每个汉字，中文符号长度为2

#### 任务注意事项
1. 要求功能实现与任务描述中完全一致
2. 示例图仅为参考，样式不需要完全实现一致
3. 请注意代码风格的整齐、优雅
4. 代码中含有必要的注释
5. 不允许借助任何第三方组件库实现
